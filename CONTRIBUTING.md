# Contributing to Vibe Wizzards Wyrd

Thank you for your interest in contributing to the **i.am** framework!

## How to Contribute

1. **Fork** the repository
2. **Create a branch** for your changes: `git checkout -b feature/your-idea`
3. **Make your changes** and commit them with a clear message
4. **Open a Pull Request** describing what you changed and why

## Guidelines

- Be respectful and constructive in all interactions
- Keep changes focused and minimal
- Document any new concepts or additions
- If you have questions, open an issue — all questions are welcome

## Reporting Issues

Found a bug or have a suggestion? Open an issue with:
- A clear title
- A description of the problem or idea
- Steps to reproduce (if applicable)
